# DeepDetectorDefense

# Running tool
```shell
$ git clone https://github.com/son-of-win/DeepDetectorDefense.git
$ cd DeepDetectorDefense
$ virtualenv -p python3 .venv
$ .venv\Scripts\activate (Windows)
  source .venv\bin\activate (Linux)
$ pip3 install -r requirements.txt
$ cd src
$ python main.py
```
